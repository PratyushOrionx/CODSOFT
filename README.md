# CODSOFT - Data Science Internship

This repository contains the projects completed as part of my Data Science Internship at CodSoft.

## Projects

### 1. Titanic Survival Prediction
Built a machine learning model to predict whether a passenger survived the Titanic disaster based on passenger information.

**Tools Used:** Python, Pandas, Matplotlib, Scikit-learn

### 2. Iris Flower Classification
Built a classification model to classify Iris flowers into different species based on their sepal and petal measurements.

**Tools Used:** Python, Pandas, Matplotlib, Scikit-learn

### 3. Sales Prediction Using Python
Analyzed advertising data and built a regression model to predict sales based on TV, Radio, and Newspaper advertising expenditure.

**Tools Used:** Python, Pandas, Matplotlib, Scikit-learn
